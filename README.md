# Portfolio

A modern personal portfolio website built with Next.js, React, and Tailwind CSS, showcasing projects, skills, and contact information. Designed to be fully responsive, visually appealing, and easy to maintain.

### Demo

A live demo of the portfolio can be hosted on Vercel or any static hosting service that supports Next.js.

[portfolio-kovaleva.vercel.app](https://portfolio-kovaleva.vercel.app)

### Features

- Fully responsive layout with mobile-first design.

- Smooth scroll navigation for in-page sections.

- Interactive skill cards with hover effects.

- Project showcase grid with image overlays and external links.

- Contact section with social links and email.

- Modern UI using glassmorphism and gradient elements.

- SEO-friendly with semantic HTML and accessible components.

### Tech Stack

- Next.js 16.1.6 – Server-side rendering and static site generation.

- React 19.2.3 – Component-based UI library.

- Tailwind CSS 4 – Utility-first styling framework.

- React Icons 5.6.0 – Icon library for React components.

- TypeScript 5 – Static typing for safer and more maintainable code.

- ESLint 9 – Code linting and formatting.

### Available Scripts

- `npm run dev` – Starts the development server.

- `npm run build` – Builds the application for production.

- `npm run start` – Starts the production server after build.

- `npm run lint` – Runs ESLint to check code for errors and style issues.
